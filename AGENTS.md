# AGENTS.md

## Project Overview / 项目概述

This is a personal photography portfolio website for Z, a Sydney-based photographer and videographer.

这是一个个人摄影作品集网站，主要用于展示摄影作品、毕业照拍摄、个人写真、商拍服务和活动拍摄服务。

## Working Rules / 工作规则

Before making major changes:
1. Read `AGENTS.md` and `PROJECT_BRIEF.md` first.
2. Inspect the project structure.
3. Give a brief plan.
4. Then implement the code.
5. Run build or relevant checks after coding.
6. Fix errors before finishing.

在进行大修改前：
1. 先阅读 `AGENTS.md` 和 `PROJECT_BRIEF.md`；
2. 先检查项目结构；
3. 简短说明计划；
4. 再开始写代码；
5. 写完后运行 build 或相关检查；
6. 如果有报错，先修复再结束。

## Technical Stack / 技术栈

Preferred:
- Vite
- React
- Tailwind CSS
- Static frontend only

Avoid:
- Backend
- Database
- Login system
- Payment system
- Unnecessary complexity

## Design Direction / 设计方向

The website should feel:
- Clean
- Professional
- Modern
- Premium but not too flashy
- Suitable for photography and commercial clients

网站风格应该是：
- 干净
- 专业
- 现代
- 有质感但不要太花
- 适合摄影作品展示和商拍客户浏览
