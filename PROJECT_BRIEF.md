# PROJECT_BRIEF.md

## 1. Project Summary / 项目摘要

Build a static personal photography portfolio website for Z, a Sydney-based photographer and videographer.  
The site should present selected photography work and clearly communicate available services to potential clients.

为 Z 制作一个静态个人摄影作品集网站。网站需要展示摄影作品，并清晰介绍可提供的拍摄服务，方便潜在客户快速了解风格与业务范围。

## 2. Primary Goals / 核心目标

1. Showcase photography work in a polished, visually compelling way.  
2. Present four service categories:
   - Graduation photography
   - Personal portraits
   - Commercial photography
   - Event photography
3. Create a professional, clean, premium visual impression.  
4. Provide an excellent mobile browsing experience.  
5. Keep the implementation simple, static, and easy to deploy on Vercel.

## 3. Target Audience / 目标用户

- Individuals looking for graduation photos or personal portraits
- Brands or businesses seeking commercial photography
- Clients planning events who need photography coverage
- Visitors who want to quickly understand Z's style, services, and contact path

## 4. Scope / 项目范围

### In Scope / 包含内容

- Static portfolio website
- Responsive layout with strong mobile support
- Hero section
- Portfolio/gallery section
- Services section
- About/intro section
- Contact or inquiry call-to-action section
- Navigation and footer
- Vercel-ready frontend project

### Out of Scope / 不包含内容

- Backend
- Database
- Login/authentication
- Admin panel
- Payment system
- Booking system with persistence
- Complex CMS integration

## 5. Preferred Tech Stack / 推荐技术栈

- Vite
- React
- Tailwind CSS
- Static frontend only

## 6. Design Direction / 设计方向

The site should feel:
- Clean
- Professional
- Modern
- Refined
- Premium without being flashy

Suggested visual qualities:
- Generous whitespace
- Strong photography-first layouts
- Neutral color palette
- Elegant typography
- Subtle motion only where it improves polish
- Clear hierarchy for services and contact actions

## 7. Experience Requirements / 体验要求

### Desktop
- Large, immersive image presentation
- Clear navigation
- Balanced spacing and readable content

### Mobile
- Fast, easy scrolling
- Images remain prominent without breaking layout
- Tap targets are comfortable
- Text remains concise and readable
- Navigation works cleanly on small screens

## 8. Content Requirements / 内容需求

The site should support:
- Photographer name / brand identity
- Short introduction
- Featured works or galleries
- Service descriptions for:
  - Graduation photography
  - Personal portraits
  - Commercial photography
  - Event photography
- Contact CTA
- Optional social links

## 9. Quality Bar / 质量标准

The finished project should:
- Build successfully with the project build command
- Be responsive across common screen sizes
- Match the intended professional and premium tone
- Avoid unnecessary dependencies and complexity
- Be suitable for deployment to Vercel without backend configuration

## 10. Delivery Expectations / 交付要求

When implementation begins later:
1. Review `AGENTS.md` and this brief first.
2. Inspect the existing project structure.
3. Provide a short plan before coding.
4. Implement the site with the preferred stack where practical.
5. Run a build check after coding.
6. Fix any build issues before finishing.
7. Explain how to deploy the completed site to Vercel.
